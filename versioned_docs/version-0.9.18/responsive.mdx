## What is Responsive Design

A responsive design is a web design that will automatically adjust for different screen sizes and viewports.


<img src="/img/doc_images/responsive_example.png" width='600'/>  

For example, when the screen height is small, a responsive bar chart will change from circular layout (a) to linear layout (b) to improve space efficiency.

## How to Implement Responsive Design In Gosling

```javascript
{
    "responsiveSize": {"width": false, "height": true},
    "views": [
    {
      "layout": "circular", // by default, we use a circular layout
      "responsiveSpec": [
        { 
         // the layout will change to linear 
         // when the "aspectRatio" is larger than 1.5 (i.e., a wider screen)
          "spec": {"layout": "linear"}, 
          "selectivity": [
            {"measure": "aspectRatio", "operation": "GT", "threshold": 1.5}
          ]
        }
        "tracks": [
            /** define your tracks here **/
        ]
}
```

**Three propoeries are commonly used to create responsive design in Gosling**:
- `responsiveSize`: 
    Whether the width and the height of the visualization will change with the screen size.
    ```boolean | {width: boolean, height: boolean}```
    
- `responsiveSpec`:
    An array that defines how certain Gosling spec will change with the screen size.
    ```javascript
    Array<{
        spec: SPEC,
        selectivity: SELECTIVITY
    }>
    ```
